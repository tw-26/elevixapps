Getting started

Install Elevix Apps on your monday.com account and add it to your boards in a few steps.

1 Install from the marketplace

Open the monday.com marketplace and search for Elevix Apps. Click Add to your account and approve the permissions. Only account admins can install apps for the whole workspace.

Tip: If you don’t see the app, ask your admin to install it from the marketplace first.

2 Add the app to a board

Open the board where you want to use Elevix Apps. Click the + button (or Integrations / Apps in the board menu), find Elevix Apps, and add it. You can add it at the board level or inside a specific item.

3 Configure and use

After adding the app, open it from the board. Follow the in-app prompts to connect any external services (if needed) and choose which columns or actions to use. Settings are saved per board.

Need help?

If you run into issues or have questions, visit our Support page for contact options and resources.
