Documentation

Clone Subitems using board view

Use board view to choose a source item and target items, then clone all subitems in one flow. This is the fastest way to apply the same task structure across multiple items.

Clone Subitems using Board Item Menu

Clone directly from the board item menu when you are already working at the item level. It is a quick option for applying subitems without leaving your current context.

Create and Apply Templates using board view

Save reusable subitem structures as templates from board view and apply them whenever needed. This helps keep recurring workflows consistent and saves setup time.

Apply Subitems Template using Board Multi Item Menu

Apply a saved template to multiple items at once from the board multi item menu. Use this when you need to roll out the same checklist across many items in one action.
