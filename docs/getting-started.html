Documentation

Everything you need to use Subitems Multiplier on your monday.com boards.

Introduction

Subitems Multiplier lets you create, clone, and apply subitems in bulk on monday.com. Instead of manually adding the same subitems to many items, you can:

  • Clone subitems from one item to one or many target items
  • Save subitems as templates (titles and column values) and reuse them on any item
  • Preview what will be created before applying changes

Usage is tracked per account per month. See the Pricing page for plan limits.

1 Getting started

Install Subitems Multiplier from the monday.com marketplace, then open the board where you want to use it.

  • Open your board and add the app via the + button or Integrations / Apps
  • Find Subitems Multiplier and add it to the board
  • Open the app from the board to start cloning or working with templates
Tip: If you don’t see the app, ask your account admin to install it from the marketplace first.

2 Cloning subitems

Clone all subitems from one item to one or multiple target items in a few steps.

  1. Select the source item. This is the item whose subitems you want to copy.
  2. Select target items. Choose one or several items where the subitems will be created.
  3. Preview. The app shows you exactly which subitems will be created (titles and column values). Review the list before applying.
  4. Apply. Confirm to create the subitems. Each new subitem counts toward your monthly usage.

Example: cloning 7 subitems to 6 items creates 42 subitems in one action.

3 Templates

Templates let you save a set of subitems (titles and column values) and reuse them on any item, anytime.

Saving a template

From an item that has the subitems you want to reuse, use the option to Save as template. Give the template a name. It’s stored at the board level and is independent of the original item.

Applying a template

Choose a saved template, then select one item or multiple items. You’ll see a preview of the subitems that will be created. Apply when you’re ready. Template count limits depend on your plan (see Pricing).

Managing templates

You can view, rename, or remove saved templates from within the app on the board. Templates are tied to the board where they were created.

4 Usage limits

Usage is tracked per monday.com account per calendar month.

  • Subitem creation: Each time the app creates a new subitem (via clone or template), it counts as one. Editing existing subitems does not count.
  • Templates: Your plan limits how many templates you can save (Free: 3; Pro and Business: unlimited).
  • Boards: All plans support unlimited boards.

When you reach your monthly subitem limit, you won’t be able to create more until the next month or after upgrading. The app will show a clear message when you’re near or at your limit.

Best practices

  • Use templates for recurring workflows. If you use the same subitem structure often (e.g. campaign phases, onboarding steps), save it as a template once and apply it whenever needed.
  • Preview before applying. Always check the preview so you know exactly which subitems will be created and on which items.
  • Name templates clearly. Use descriptive names (e.g. “Campaign launch checklist”, “Client onboarding”) so your team can find them quickly.
  • Stay within your plan. Keep an eye on your monthly subitem usage so you don’t hit the limit mid-month. Upgrade if you need more.
